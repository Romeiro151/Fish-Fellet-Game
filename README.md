# Fish-Fellet-Game
This project was developed within the scope of an Object-Oriented Programming (OOP) course using Java. The main objective was to apply fundamental OOP principles to design and implement a structured and modular application.

The project follows core OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction. The system is organized into multiple classes representing different entities, each responsible for specific behaviors and attributes. This approach ensures code reusability, maintainability, and scalability.

The architecture of the project separates responsibilities clearly, promoting clean code practices and modular design. Classes interact through well-defined interfaces.
